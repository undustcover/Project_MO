## 基础录入数据

### excel表录入字段说明

#### 一、进度excel表

##### 1.实施数据

| 字段名称     | 字段属性 | 字段说明                                                     | 填报单位 |
| :----------- | -------- | ------------------------------------------------------------ | -------- |
| 井号         | 文本     | 生产时效与进度所发生的井场井号                               | 项目部   |
| 工况名称     | 文本     | 根据生产进度时间里程碑时间所区分的各个独立结算单元           | 项目部   |
| 工况一级时效 | 单选     | 生产时效、非生产时效                                         | 项目部   |
| 工况二级时效 | 单选     | 动拆搬安、进尺工作时间、固井工作时间、电测时间、辅助工作时间、组织停工时间、修理时间、复杂时间、故障时间、自然灾害停工时间、其他停工时间、试修时间 | 项目部   |
| 工况三级时效 | 单选     | 动迁、纯钻、起下钻、换接划、循环、其他、拆卸、搬家、安装、油气水侵复杂、井塌缩径、固井复杂、测井故障、卡钻故障、落物故障、其他故障、试油、试气、修井 | 项目部   |
| 开始时间     | 日期     | hh:mm                                                        | 项目部   |
| 结束时间     | 日期     | hh:mm                                                        | 项目部   |
| 负责人       | 文本     | 负责人姓名                                                   | 项目部   |
| 日期         | 日期     | yy:mm:dd                                                     | 项目部   |

##### 2. 计划数据

| 字段名称       | 字段属性 | 字段说明        | 填报单位 |
| -------------- | -------- | --------------- | -------- |
| 合同建井周期   | 时间：h  | 合同中的约定：h | 运管部   |
| 合同动迁周期   | 时间：h  | 合同中的约定：h | 运管部   |
| 合同搬安周期   | 时间：h  | 合同中的约定：h | 运管部   |
| 合同钻井周期   | 时间：h  | 合同中的约定：h | 运管部   |
| 合同完井周期   | 时间：h  | 合同中的约定：h | 运管部   |
| 合同非生产时间 | 时间：h  | 合同中的约定：h | 运管部   |
| 合同周期       | 时间：h  | 合同中的约定：h | 运管部   |

##### 3. 聚合数据

| 字段名称       | 字段属性 | 字段说明                                                 | 统计关联 |
| -------------- | -------- | -------------------------------------------------------- | -------- |
| 项目生产时效   | 时间：h  | 项目所有生产时效之和                                     |          |
| 项目非生产时效 | 时间：h  | 项目所有非生产时效之和                                   |          |
| 任务生产时效   | 时间：h  | 项目下单个节点任务的生产时效之和                         |          |
| 任务非生产时效 | 时间：h  | 项目下单个节点任务的非生产时效之和                       |          |
| 建井周期       | 时间：h  | 单个井号相同的所有时效之和                               |          |
| 搬安周期       | 时间：h  | 单个井号下所有工况名称有搬安的时效之和                   |          |
| 钻井周期       | 时间：h  | 单个井号下除了搬安周期、完井周期、试油周期之外的时效之和 |          |
| 完井周期       | 时间：h  | 单个井号下所有工况名称有完井的时效之和                   |          |
| 试油周期       | 时间：h  | 单个井号下所有工况名称有试油、试气任意一个词组的时效之和 |          |

#### 二、成本excel表格

##### 1. 实施数据

| 字段名称               | 字段属性       | 字段说明                                                     | 填报单位 |
  | ---------------------- | -------------- | ------------------------------------------------------------ | -------- |
  | 项目名称               | 文本           | 表明属于哪一个项目                                           | 项目部   |
  | 任务名称               | 文本           | 表明属于项目中哪一个里程碑事件                               | 项目部   |
  | 成本要素组-1级名称     | 单选           | 1.项目直接成本；2.项目间接成本                               | 项目部   |
  | 成本要素组-2级名称     | 单选           | 1.项目直接成本-人工费；2.项目直接成本-服务费；3.项目直接成本-材料费；4.项目直接成本-设备费；5.项目间接成本-分摊费；6.项目间接成本-计提费。 | 项目部   |
  | 成本要素组-3级名称     | 文本           | 格式为：项目直接成本-XX费-XX费                               | 项目部   |
  | 成本计划细分项-4级名称 | 文本           | 选填                                                         | 项目部   |
  | 单位                   | 文本           | 项目成本的计价单位                                           | 项目部   |
  | 单价                   | 金额           | 项目成本的金额                                               | 项目部   |
  | 数量                   | 数字           | 项目成本的数量                                               | 项目部   |
  | 总成本                 | 项目成本的总价 | 由单价*数量得出                                              | 项目部   |
  | 收入                   | 金额           | 选填，如果有收入，应该写明                                   | 项目部   |

##### 2. 计划数据

| 字段名称     | 字段属性 | 字段说明                                   | 填报单位 |
| ------------ | -------- | ------------------------------------------ | -------- |
| 项目总预算   | 金额     | 项目的成本预算                             | 运管部   |
| 任务总预算   | 金额     | 项目中单个里程碑事件任务的预算             | 运管部   |
| 任务人员预算 | 金额     | 项目中单个里程碑事件任务中人员成本预算     | 运管部   |
| 任务材料预算 | 金额     | 项目中单个里程碑事件任务中材料成本预算     | 运管部   |
| 任务设备预算 | 金额     | 项目中单个里程碑事件任务中设备成本预算     | 运管部   |
| 任务服务预算 | 金额     | 项目中单个里程碑事件任务中服务采购成本预算 | 运管部   |

##### 3.聚合数据

| 字段名称       | 字段属性 | 字段说明                                                     | 统计关联 |
| -------------- | -------- | ------------------------------------------------------------ | -------- |
| 项目总成本     | 金额     | 相同项目名称字段下所有成本之和                               |          |
| 任务总成本     | 金额     | 单个相同项目名称字段下，相同任务名字段下的所有成本之和       |          |
| 任务人员成本   | 金额     | 单个相同项目名称字段下，相同任务名字段下，成本要素组-2级名称所有为项目直接成本-人工费的成本之和 |          |
| 人员材料成本   | 金额     | 单个相同项目名称字段下，相同任务名字段下，成本要素组-2级名称所有为项目直接成本-材料费的成本之和 |          |
| 人员设备成本   | 金额     | 单个相同项目名称字段下，相同任务名字段下，成本要素组-2级名称所有为项目直接成本-设备费的成本之和 |          |
| 人员设服务成本 | 金额     | 单个相同项目名称字段下，相同任务名字段下，成本要素组-2级名称所有为项目直接成本-服务费的成本之和 |          |

##### 重点项目跟踪：

一、设置重点项目后自动带出来的字段

1. 项目区域，来自市场国别字段。
2. 合同编号，来自合同编号字段。
3. 执行主体，来自执行主体字段
4. 钻机编号，来自队伍编号（国工编号）字段
5. 项目期限，来自于合同周期，开始时间~结束时间

二、新建重点项目导入excel表。

1. 合同编号，需要校对是否有合同编号，如果没有抛出错误，提示XXX合同编号没有录入，并拒绝导入excel表格信息。
2. 项目名称：文本
3. 工作量（口）：数字
4. 项目实时进度：文本
5. 首井开钻时间：时间（YYYY:MM:DD:hh:mm）
6. 重点关注事项：文本
7. 已完成价值工作量：文本
8. 今年预计完成工作量：文本
