## 雷达图

现在要设定考核指标6维雷达图：
基础数据有：搬安时间（h）；钻进时间（h）；中完时间（h）；生产时间（h）；非生产时间（h）；进尺工作时间（h）；非进尺工作时间（h）；纯钻时间（h）；起下钻时间（h）；循环时间（h）；换接划时间（h）；其他时间（h）合同规定钻进时间（h）；合同规定中完时间（h）；合同规定生产时间（h）；合同规定非生产时间（h）。

#### 基础数据获取方式：

1. 生产时间：之前的模块已经进行计算，名称为生产时效
2. 合同生产时间：用户在网页页面录入
3. 非生产时间：之前的模块已经进行计算，名称为非生产时效。
4. 实际中完时间：同一井号下，工况二级时效字段下所有固井工作时间与电测工作时间之和。
5. 纯钻时效：之前的模块已经进行计算
6. 起下钻时间：同一井号下，工况三级时效字段下所有起下钻之和
7. 进尺工作时间：同一井号下，工况二级时效字段下所有进尺工作时间之和
8. 

#### 考核指标

1. 生产时间达成率（%）：公式为：|（钻进周期内的生产时间+完井周期内的生产时间-合同钻进周期+合同完井周期）/（实际钻进周期+实际完井周期）|×100%。考核标准：1.优秀+：1%以内是优秀+，得5分，2.优秀：1<B≤2%，得4分；3.良好+：2%<B≤6%，得3分；4.良好：6%<B≤10%，得2分，5.一般：10%<B≤20%，得1分，不合格：>20%不合格，得0分。

2. ==非生产时效溢价率（%）：计算公式为两个维度的权重相加
   2.1 A.非生产时效占比：(非生产时间/(生产时效+非生产时效)*100%
    评分:A≤0.3%得分为5分；0.3%<A≤0.5%得分为4分；0.5%<A≤0.8%得分为3分；0.8%<A≤1.0%得分为2分；1.0%<A≤2%得分为1分；A>2%得分为0分。
    2.2 B.非生产时间合同符合率：非生产时间/合同非生产时间*100%
    评分：B≤60%得分为5分;60%<B≤70%得分为4分; 70%<B<80%得分为3分;80%<B<90%得分为2分;90%<B≤100%得分为1分;B>100%得分为0分.
    2.3 最终得分C：两者权重相加 A得分*0.35+B得分*0.65
    2.4 最终得分C的考核标准：1. 优秀+：综合得分 ≥ 4.5 分；2. 优秀：综合得分 ≥ 3.0 分；3. 良好+：2.5分 ≤ 综合得分 < 3.5分；4. 良好：1.5分 ≤ 综合得分 < 2.5 分；3.一般：0.5 分 ≤ 综合得分 <1.5 分；4.不合格：综合得分 < 0.5 分==

3. ==中完时间达成率（%）：公式为：A3=实际中完时间 ÷ 合同中完时间：1. 优秀+：A3≤60%，得5分；2. 优秀：60%<A3≤70%，得4分；3. 良好+：70%<A3≤80%，得3分；4. 良好：80%<A3≤90%，得2分；5. 一般：90%<A3≤100%，得1分；6.不合格：A3>100%，得0分。==

4. ==纯钻时效：纯钻时间/(钻井周期内的生产时间+非生产时间)×100%：1. 优秀+: ≥50%，得5分，2. 优秀 ≥40%，得4分；3.良好+: ≥35%，得3分；4. 良好 ≥30%，得2分；5.一般: ≥20%，6.不合格: <20%==

5. 起下钻时效：起下钻时间/进尺工作时间×100%：1. 优秀+: ≤7%；2. 优秀: ≤10%；3. 良好+：≤12%；4. 良好: ≤15%；5. 一般: ≤20%；6. 不合格: >20%

6. 合同时间利用率：考核标准：

   6. 1 A1：搬安时间利用率：公式为：搬安周期/合同搬安周期*100%， 评分：B≤60%得分为5分; 60%<B≤70%得分为4分;70%<B≤80%为3分；80<B≤90%得分为2分；90<B≤100%得分为1分；B>100%得分为0分。

      6.2 B1: 钻井时间利用率：公式为：钻井周期-合同钻进周期/合同钻进周期*100%，评分为：A≤0.5%得分为5分；0.5<A≤1.0%得分为4分;1<A≤1.5%得分为3分；1.5<A≤2.0%得分为2分；2.0%<A≤2.5%得分为1分；A>2.5%得分为0分。

      6.3  C1：完井时间利用率：公式为：完井周期-合同完井周期/合同完井周期*100%，评分为：A≤0.5%得分为5分；0.5<A≤1.0%得分为4分;1<A≤1.5%得分为3分；1.5<A≤2.0%得分为2分；2.0%<A≤2.5%得分为1分；A>2.5%得分为0分。

     6.4 最终得分D1：A×0.125+B×0.625+C×0.25

     6.5  1. 优秀+：综合得分 ≥ 4.5 分；2. 优秀：3.5 分 ≤ 综合得分 <4.5 分；3.良好+：2.5 分 ≤ 综合得分 <3.5 分；4.良好：1.5 分 ≤ 综合得分 <2.5 分；5.一般：0.5 分 ≤ 综合得分 <1.5 分；6.不合格：综合得分 < 0.5 分
