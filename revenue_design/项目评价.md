## 详细计算说明

### 基础指标计算

**1. 进度偏差指数 (SVI)**

python

```
# 对于每个任务
计划工时 = 计划完成时间 - 计划开始时间
延误工时 = MAX(0, 实际完成时间 - 计划完成时间)

# 项目级计算
总计划工时 = SUM(所有任务的计划工时)
总延误工时 = SUM(所有任务的延误工时)
SVI = (总计划工时 - 总延误工时) / 总计划工时
```



**2. 成本绩效指数 (CPI)**

python

```
CPI = SUM(合计价值工作量) / SUM(任务成本)
# 注意：CPI > 1 表示成本效益好，CPI < 1 表示成本超支
```



**3. 收入健康度 (RHI)**

python

```
收入达成率 = SUM(已确认金额) / SUM(收入计划)
回款效率 = SUM(收现金额) / SUM(已确认金额)
RHI = 收入达成率 × 回款效率
```



### 高级指标计算

**4. 现金流贴现指数 (DCFI) - 运用DCF模型**

python

```
# 按时间周期(如月度)聚合现金流
对于每个周期t:
    CF_t = 该周期收现金额 - 该周期任务成本
    PV_t = CF_t / (1 + r)^t  # r为年贴现率，t为年数

总现值 = SUM(所有周期的PV_t)
总成本 = SUM(任务成本)
DCFI = 总现值 / 总成本
```



**5. 价值效率指数 (VEI) - 运用变异系数调整**

python

```
# 计算基础效率
基础效率 = SUM(合计价值工作量) / SUM(实际工作量)

# 计算稳定性调整因子
每米收入数组 = [每个任务的每米收入]
均值 = AVERAGE(每米收入数组)
标准差 = STDEV(每米收入数组)
变异系数CV = 标准差 / 均值

VEI = 基础效率 / (1 + CV)
```



**6. 计划校准分数 (PCS) - 运用欧几里得距离**

python

```
# 时间预测误差
时间误差 = ABS(实际完成时间 - 计划完成时间) / (计划完成时间 - 计划开始时间)
平均时间误差 = MEAN(所有任务的时间误差)

# 收入预测误差  
收入误差 = ABS(已确认金额 - 收入计划) / 收入计划
平均收入误差 = MEAN(所有任务的收入误差)

PCS = 1 - SQRT(0.6×平均时间误差² + 0.4×平均收入误差²)
```

### 异常模式识别

1. **高SVI低PCS**: 进度良好但预测不准，可能靠资源堆砌达成
2. **高CPI低VEI**: 成本控制好但效率低，可能牺牲了质量或范围
3. **高RHI低DCFI**: 收入达标但现金流差，回款周期可能过长

# 项目运气评估

以下是整合高级指标与基础关键指标的总结列表，结合雷达图多维度展示需求，明确各指标的核心信息：

1. **时间波动稳定性指数**
   - 核心含义：衡量收入确认、收现时间节点的波动程度，反映外部干扰（如客户流程）对时间管理的影响。
   - 关联基础数据字段：收入确认时间差（24）、收现时间差（25）。
   - 雷达图维度：「时间节点稳定性」。
   - 关键解读：取值 [0,1]，>0.8 为稳定（外部干扰小），<0.3 为剧烈波动（受外部因素影响大）。
2. **工作量收入弹性指数**
   - 核心含义：量化实际工作量变动对合计价值工作量的敏感程度，体现资源投入的收益杠杆效应。
   - 关联基础数据字段：实际工作量（9）、合计价值工作量（13）、历史同类任务数据。
   - 雷达图维度：「资源投入 - 收益敏感性」。
   - 关键解读：>1 表示工作量对收入拉动强（高附加值），<0 表示投入与收益反向（如单价骤降）。
3. **综合绩效偏离度指数**
   - 核心含义：整合进度、成本、收入三大维度与计划的偏差，衡量项目整体与预期的偏离程度。
   - 关联基础数据字段：计划 / 实际开始 / 完成时间（5-8）、任务成本（14）、计划成本、收入偏差（23）、收入计划（15）。
   - 雷达图维度：「整体目标偏离度」。
   - 关键解读：取值 [0,1]，>0.8 为接近预期（内外部协调），<0.3 为严重偏离（多重不利因素）。
4. **成本收入协同指数**
   - 核心含义：衡量成本投入与已确认金额的增长协同性，反映资源转化效率。
   - 关联基础数据字段：任务成本（14）、已确认金额（17）、上期同类任务数据。
   - 雷达图维度：「成本 - 收入协同效率」。
   - 关键解读：取值 [0,1]，>0.7 为协同增长（外部环境有利），<0.3 为协同性差（如材料涨价冲击）。
5. **回款率（基础指标）**
   - 核心含义：收现金额与合同总金额的比例，反映资金回收效率。
   - 关联基础数据字段：收现金额（19）、合同总金额（20）（字段 22 直接提供）。
   - 雷达图维度：「资金回收效率」。
   - 关键解读：越高说明资金回流越及时，受客户支付能力等外部因素影响。
6. **每米收入与每米成本（基础指标）**
   - 核心含义：单位工作量（米）的收益与成本，体现细分维度的投入产出比。
   - 关联基础数据字段：合计价值工作量（13）、任务成本（14）、单位为 “米” 的实际工作量（9、10）（字段 28、29 直接提供）。
   - 雷达图维度：「单位产出 / 成本效益」。
   - 关键解读：每米收入 - 每米成本差值越大，单位资源的盈利性越强。

**雷达图应用说明**：

- 以上 6 项指标构成雷达图的 6 个维度，通过各维度数值的 “多边形面积” 直观反映项目综合状态。
- 若多边形整体偏向外侧（各指标值高），说明项目受外部有利因素（如市场稳定、客户配合）影响大（“运气好”）；若整体偏向内侧（各指标值低），则受外部不利因素（如延迟、成本上涨）冲击明显（“运气差”）。

- 若 12-17 分，结论为 “项目运气一般，外部因素正负影响均衡，需关注个别指标波动”；
- 若 6-11 分，结论为 “项目运气较差，受客户延迟、成本上涨等外部因素拖累，多项指标不达标”；
- 若≤5 分，结论为 “项目运气极差，受重大外部风险（如市场突变、客户违约）冲击，项目运行异常”。

### 6. 异常模式识别

通过指标联动分析识别以下异常模式（多为运气差的极端表现）：

1. **“高投入低回报” 模式**：工作量收入弹性指数 < 0 + 每米收入低于历史均值 20% + 收入偏差负向≥10% → 可能因材料涨价、单价骤降等外部因素导致投入浪费；
2. **“时间 - 资金双延迟” 模式**：时间波动稳定性指数 < 0.3 + 回款率 < 70% + 收现时间差远超历史均值 → 可能因客户流程混乱、资金链问题导致项目停滞风险；
3. **“全维度偏离” 模式**：综合绩效偏离度指数 < 0.3 + 3 项基础指标均≤1 分 → 受多重外部不利因素（如政策变动、市场萎缩）叠加影响，需紧急干预。
