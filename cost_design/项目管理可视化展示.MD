设计一个基于excel表格的看板+数据库高级excel项目管理日历看板与仪表盘

#### 图形化界面要求

1. 日历看板，可以调整到每个月的日历，日历主要包含24小时的“小时级"工作情况，每天要校核是否是24小时。主要显示每个工况的名称，工况的时间，工况的时效是那种。

2. 仪表盘

   2.1 成本卡组 
- 预算总成本（BAC）

- 实际成本（AC）

- 价值工作量（EV）

- 成本偏差（CV = EV – AC）

- 成本绩效指数（CPI = EV / AC）

- 完工尚需估算（ETC = (BAC – EV) / CPI）

- 完工总估算（EAC = AC + ETC）
  → 颜色阈值：CPI < 0.9 红色，0.9–1.0 黄色，>1.0 绿色
  
  2.2 进度卡片组
  
- 计划值（PV）

- 价值工作量（EV）

- 进度偏差（SV = EV – PV）

- 进度绩效指数（SPI = EV / PV）

- 关键路径天数（CPD）

- 里程碑达成率（已完成里程碑/总里程碑） → SPI < 0.9 红色预警

​     2.3 收入卡片组

- 合同总金额（Total Contract Value，TCV）

- 已确认收入（Recognized Revenue）

- 已开票未回款（AR）

- 回款率（Cash Collection / TCV）

- 收入偏差（RevCV = 已确认 – 计划确认）

- 预计总收入（EAC Revenue）

  2.4 小时级管理饼状图

- 时效统计：生产时效是多少，非生产时效是多少

- 生产时效分析：各个生产时效占比

- 非生产时效分析：各个非生产时效占比

- 进尺工作时间分析：各个进尺工作时间占比

  

  2.5三维雷达图（Radar）

- 维度：CPI、SPI、回款率。
  一眼看出“哪个维度先崩”。

  2.6双轴趋势图（Trend）

- 横轴：周/月
  左轴：累计AC、EV、PV
  右轴：累计已确认收入、已回款
  一条图同时暴露“成本超支+收入滞后”叠加风险。

  2.7预警列表

- CPI<0.9 或 SPI<0.9 或 回款率<80% 自动推顶部

#### 数据设计要求

1. 日历看板

   1.1 字段名称与要求

   - 工况名称：文本
   - 工况一级时效：单选，生产时效，非生产时效
   - 工况二级时效：联动一级时效单选，生产时效：进尺工作时间，固井工作时间，电测时间，辅助生产时间；非生产时效：组织停工时间，修理时间，复杂时间，故障时间，自然灾害停工，其它停工时间。
   - 工况三级时效：联动二级时效单选，在生产时效中，仅当二级时效为进尺工作时间时，必填，其他不填。分为：纯钻时间，取芯时间，起下钻时间，换接划时间，循环时间，其他时间。在非生产时效中，仅当二级时效为复杂时必填，其他不填，分为：井漏复杂，油气水侵复杂，井塌复杂，固井复杂，测井复杂，其他。
   
   2. 仪表盘数据：根据前面的数据设计。
   
   3. 数据库使用excel。

#### 运行要求

1. 本地运行，无需安装任何依赖，仅有excel
2. 动态更新，通过excel数据表的更新，可以直接使看板和仪表盘数据更新。
